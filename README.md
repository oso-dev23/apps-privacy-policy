** プライバシーポリシー
1. 個人情報の収集について：本アプリ「えほんきろく」は、提供する機能（本棚の管理や記録）のために、お子さまのお名前、誕生日、および絵本の読書記録を収集します。これらのデータは Supabase (データベース) に安全に保存されます。
2. 利用目的：収集したデータは、アプリ内での表示およびパーソナライズされた機能の提供のためにのみ使用されます。
3. 第三者への開示：法令に基づく場合を除き、取得した個人情報を第三者に提供することはありません。また、広告配信のために Google AdMob を使用していますが、ユーザーを特定する個人情報は送信されません。
4. データの削除：ユーザーはアプリを退会することで、保存されたすべてのデータを削除することができます。
